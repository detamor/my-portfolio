# Natanael Detamor — Engineering Portfolio

A high-performance portfolio architected for technical transparency and professional engineering standards. This repository serves as a live demonstration of modern frontend architecture, state management, and interactive UI orchestration.

## 🚀 Architectural Highlights

- **Framework:** React 18 + Vite (ESM)
- **Animation Engine:** Framer Motion (Declarative orchestration & physics-based transitions)
- **Styling:** Tailwind CSS (Utility-first system with custom configuration)
- **Typography:** Inter & Monospace (System font stacks for maximum performance)
- **Icons:** Lucide React (Tree-shakable SVG nodes)

## 🛠 Core Systems

### 1. Technical Case Study Engine
Since many of my high-impact projects are confidential enterprise systems, I developed a custom **Case Study Modal** system. This allows recruiters to analyze:
- System Architecture flows
- Database Engineering (Censored schemas)
- Performance optimization logic (Latency reduction strategies)

### 2. Kinetic UI Components
- **Dynamic Projects Grid:** Utilizes 3D perspective transforms mapped to cursor coordinates.
- **Precision Hero Section:** Implements digital dust particle systems and structural tech nodes.
- **Responsive Fluidity:** Fully optimized for high-DPI displays and mobile breakpoints.

## 📁 Project Structure

```bash
src/

│   ├── CaseStudyModal.jsx  # Technical documentation engine
│   ├── HeroSection.jsx     # High-impact entry point
```

## ⚙️ Local Development

1. **Clone & Install**
   ```bash
   git clone https://github.com/detamor/my-portfolio.git
   cd my-portfolio
   npm install
   ```

2. **Run Dev Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
